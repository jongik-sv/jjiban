## 테스트 케이스

### TC-001: Hierarchical LR 레이아웃

| 항목 | 내용 |
|------|------|
| 목적 | 그래프가 왼쪽→오른쪽 방향으로 정렬되는지 확인 |
| 사전조건 | 의존관계가 있는 Task 데이터 존재 |
| 절차 | 1. 그래프 모달 열기 2. 노드 위치 확인 |
| 예상결과 | 독립 노드는 왼쪽, 의존 노드는 오른쪽에 배치 |

### TC-002: 인터랙션 (드래그/줌/팬)

| 항목 | 내용 |
|------|------|

| 사전조건 | 그래프 모달 열린 상태 |
| 절차 | 1. 노드 드래그 2. 마우스 휠 스크롤 3. 배경 드래그 |
| 예상결과 | 각 인터랙션이 정상 동작 |

### TC-003: 모달 열기/닫기

| 목적 | 그래프 모달 열기/닫기 동작 확인 |

| 절차 | 1. 그래프 버튼 클릭 2. 모달 닫기 버튼 클릭 |

### TC-004: Task 의존관계 표시

| 항목 | 내용 |
|------|------|

| 사전조건 | depends 필드가 있는 Task 존재 |
| 절차 | 1. 그래프 모달 열기 2. 노드/엣지 개수 확인 |
| 예상결과 | Task 수 = 노드 수, 의존관계 = 엣지 수 |

### TC-005: 카테고리별 색상
